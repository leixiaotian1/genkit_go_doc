# Genkit Go 中文文档

本仓库使用 Gemini PRO 模型将 Firebase Genkit Go 的英文文档翻译为中文。

## 项目概述

Firebase Genkit Go 是一个用于构建 AI 应用的 Go 框架。本文档提供了框架的完整中文翻译，帮助中文开发者更好地理解和使用该框架。
